## Question
Round-01 (QUALIA_LAB heavy-arm, `dak_evolution`, **N=1 seed/level**, **confounded env-RNG**) reported a regime-dependent, non-monotone sweep — winners **[Z, N, Z, N]** for [L0, L1, L2, L3] — from which two claims were read (established.md **F6**/**F8**):
- **strong:** "the gap's advantage grows monotonically with complexity" → already **WOUNDED** (the sweep is non-monotone).
- **weak:** "the gap wins in threat (predators, L1) / deprivation (night, L3) regimes" → said to **stand**.

The science's own skeptic (**D-DAK-RNG**) flagged two flaws blocking the per-regime causal story: (1) an **env-RNG divergence confound** (per-gen draw counts differ by level ⇒ the base world silently differs across levels), and (2) **N=1** (no significance). This run fixes both and re-tests.

## Method (what changed vs round-01)
1. **Confound fixed (D-DAK-RNG(1)).** The environment is generated by a purpose-keyed splitmix64; the base layout (lights, food, **and predators**) is keyed by `(seed, gen)` **only** — level-independent — and every entity is drawn at every level (predators drawn but inactive at L0). Only the *dynamics* (predator activity L1+, light motion L2+, night L3) are gated. **Proven, not asserted:** `someone --selftest` hashes the gen-0 base layout for all four levels and asserts byte-identity (passes). The clean **L2↔L3** night A/B is preserved.
2. **Powered (D-DAK-RNG(2)).** `--ensemble 24` replicas/level (replica r uses seed+r), vs N=1. Per-replica final Δfit = normal_fit − zombie_fit; an exact **two-sided sign test** licenses a winner only at p<0.05 (else TIE).
3. **Deterministic.** Whole ensemble is byte-reproducible from the top seed (determinism proven byte-identical ×3 on the golden; structural — no wall-clock, no float atomics).
4. **Config.** `--pop 120 --gens 70 --steps 800 --N 128 --k 32 --zombie-frac 0.5 --seed 20260705`, per level L0–L3. `steps=800` so L3's night fires (cycle 500–699). This is a **de-confounded, properly-powered re-test of the same question**, at a lighter operating point than round-01's heavy arm (N=256, pop=200) — see Caveats.

## Verdict — what reproduces, what is sharpened, what is overturned

- **Strong form ("advantage grows monotonically with complexity"): NOT SUPPORTED — reconfirmed, now corpus-grade.** Round-01 wounded it with N=1; the wound survives de-confounding + n=24 (mean Δfit is non-monotone; no monotone trend). This is now a properly-powered negative, not a single-seed artifact.

- **Weak form ("gap wins in threat/deprivation regimes L1/L3"): NOT STATISTICALLY SUPPORTED at n=24.** L1 (predators) leans normal (14/10) but n.s. (p=0.27); L3 (night) is even/slightly zombie (11/12). There is a *directional* hint of a normal advantage under predators (L1) only, well short of significance. As a *significant* claim the weak form **does not stand** here — a stronger statement than round-01's "it stands."

- **"zombie wins L0 / L2": OVERTURNED.** L0 leans zombie (16/8) but misses significance (p=0.076); L2 is a dead 12/12 tie. Neither is a corpus-grade zombie win. Round-01's L0/L2 zombie-wins were **N=1 sampling + the env-RNG confound**, not robust regime effects.

- **The headline overturn:** round-01's entire per-regime winner structure **[Z, N, Z, N] collapses to [T, T, T, T]** under de-confounding + n=24. The DAK evolutionary setup has such large **across-seed variance** (founder effects: in each replica one class stochastically fixes) that a single seed yields a "winner" that is essentially an amplified coin flip. Averaged fairly, the self-model gap is **≈ fitness-neutral in every tested regime** on this instrument. This is a functional-neutrality result, and a direct vindication of D-DAK-RNG.

## Caveats (honest)
- **Config differs from round-01's heavy arm** (N=128/pop=120 vs N=256/pop=200). This is a de-confounded *re-test of the question*, not a bit-parity replay. A smaller pop/N raises founder-effect variance, which *could* inflate ties — but the golden at N=256 shows the same high-variance/no-clear-winner behavior, so the qualitative conclusion (per-level winners are not robust; strong form fails) is cross-config. A full **N=256, n≥20** confirmation is the natural (expensive, ~2–3 h) follow-up if the science wants heavy-arm parity.
- This instrument's `viability` scalar is a **loose** C3 carrier (established.md F8 note); selection here runs through the behavioral-survival (death) channel. We report survival/alive-counts + Δfit as the signal, not `viability`.
